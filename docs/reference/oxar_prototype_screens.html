About this demo

Clickable prototype of the OX.ar Embryo Module enhancements — for showing users, not production. All patients/specimens are synthetic (SAMPLE / DEMO / PROTO).

  • Treatment Planning — change the treatment type to see fields and cycle tabs show/hide, and the plan steps update. Driven by the per-type templates.
  • Three-Point Check — tick the three identifiers and pick an independent witness to unlock the procedure.
  • Cryo Tanks — fill gauges + status; click a tank for the canister/goblet/straw layout and inventory.
